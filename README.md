# Mecab::Syllable

日本語の文章を音節で分割して扱うことができます。

使用例: 俳句や短歌の推敲として、音数を数える

## Installation

Add this line to your application's Gemfile:

    gem 'mecab-syllable'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install mecab-syllable
